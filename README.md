## 📌 Project Overview

This project is an end-to-end Data Analytics solution built using **Python, PostgreSQL, Streamlit, and Plotly**.

The dashboard analyzes Flipkart product data to uncover insights related to:

* Product pricing
* Discounts
* Brand performance
* Category distribution
* Product search and exploration

The project demonstrates a complete analytics workflow, from data ingestion and storage to visualization and deployment.

## 📈 Key Insights

* Average discount across products is approximately **40%**.
* Jewellery products represent a significant portion of the inventory.
* Product prices are concentrated in the lower and mid-price ranges.
* Certain categories offer substantially higher discounts than others.
* Thousands of products across multiple brands are available for analysis.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/milkywayisouruniverse/FLIPKART-PRICE-ANALYSIS.git
cd FLIPKART-PRICE-ANALYSIS
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Database

Create:

```text
.streamlit/secrets.toml
```

Add:

```toml
DATABASE_URL = "YOUR_NEON_CONNECTION_STRING"
```

### Run Application

```bash
streamlit run app.py
```
